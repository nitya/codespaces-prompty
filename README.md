## Setup

Launch your browser in incognito mode for convenience.

1. Login into GitHub ‚Üí [signup](https://github.com/signup) for a free account
1. Navigate to [codespaces-prompty](https://github.com/nitya/codespaces-prompty)
1. Click `Use this template` ‚Üí then select `Create a new repository`

You should see a template instance in your profile. Navigate to it in browser.

1. Click `Code` button ‚Üí switch to `Codespaces` tab
1. Click `Create new codespace on main` ‚Üí  launches a new browser tab
1. Wait till Codespaces is ready ‚Üí  you see a VS Code IDE with terminal

This takes a few minutes. You now have a `dev container` pre-configured with required dependencies and tools for working with Prompty. Next, let's launch the workshop instruction guide:

1. Go to VS Code Terminal ‚Üí type `mkdocs serve`
1. You will see a dialog popup ‚Üí select the browser option to view
1. Wait till the page loads ‚Üí you see a website with a Prompty Workshop guide.

## Cleanup

GitHub provides a generous free _monthly_ quota for CPU and storage - but it is a finite quota. To conserve it, make sure you _Stop_ or _Delete_ your codespaces when not actively working through exercises.

- Look at [Github Settings](https://github.com/settings/codespaces) under Billing ‚Üí Usage - to see quota usage
- Look at [GitHub Codespaces](https://github.com/codespaces) ‚Üí to see active codespaces (to stop or delete them)
